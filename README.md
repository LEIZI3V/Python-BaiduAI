# 如何使用

## 1.安装库文件
* 如果已安装pip，执行pip install baidu-aip即可。 
* 如果已安装setuptools，执行python setup.py install即可。  

## 2.创建存放所要识别图片的文件夹
* 在.py文件路径下创建pictures文件夹,将所要识别的图片放进此文件夹  

## 3.直接点击执行.py文件
* 稍等片刻即可看到识别的结果
